# Glacier Curling Pro

Launch the game with any static server.

```bash
python -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

## Controls
- Drag on the sheet or use sliders to set the shot.
- Press **Space** to throw a stone.
- Hold **Shift** or the Sweep button to reduce friction.
- Press **U** to undo the last shot.
- Press **R** to reset the current end.

## Features
- Shot presets and trajectory aim assist.
- Shot history and end summaries.
- Practice mode with team selection.
